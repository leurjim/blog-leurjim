Después de la conferencia de [Gatsby](https://www.gatsbyjs.com/) 2021 y investigando como desarrollar un blog encontre un curso en [Egghead](https://egghead.io/) realizado por [Laurie Barth](https://twitter.com/laurieontech) de Crear un blog de desarrollador con Gatsby.

Gatsby es un generador de sitios estáticos basado en React y alimentado por GraphQL. Combina las mejores partes de React, Webpack, react-router, GraphQL y otras herramientas de front-end para brindar una excelente experiencia de desarrollador.

Este curso lo lleva desde la inicialización de un sitio de Gatsby a través de CLI hasta la utilización del ecosistema de Gatsby para obtener archivos MDX de su sistema. Después de abordar el sistema de archivos, optimizará para SEO, así como estilos adicionales a través de ThemeUI que admitirán características como el resaltado de sintaxis en fragmentos de código. Finalmente, aprenderá cómo procesar y renderizar imágenes de manera eficiente.

Al final del curso, tendrá un blog de desarrollador en pleno funcionamiento que servirá como base para aplicaciones de grado de producción más grandes.

## Use npm init gatsby para crear un sitio inicial de Gatsby

```Js
// Crea un sitio Gatsby
npm init gatsby
```

El sitio incluye un **package.json** con Gatsby, React e react-dominstalado, un archivo vacío gatsby-config.js y dos archivos en src/pages- index.js y 404.js.

## Mi toque personal al Blog

Después de haber desarrolado el blog pude comenzar a hacer modificaciones y darle mi toque personal. Para esto investigue otros blog como el de [Dan Abramov](https://overreacted.io/) y [Joel Hooks](https://joelhooks.com/) ambos instructores de [Egghead](https://egghead.io/).

De los cuales pude adaptar el **ThemeToggler.js**, con el cual pude desarrollar el Modo Nocturno (Dark Mode).

```js
import React from 'react'
import { useColorMode } from 'theme-ui'
import { FiMoon, FiSun } from 'react-icons/fi'
import { rgba } from 'polished'
import Button from './Button'
import colors from '../lib/colors'
import { grayscale } from 'polished'

const ThemeToggler = () => {
  const [colorMode, setColorMode] = useColorMode()

  return (
    <Button
      id="dark-mode-toggler"
      css={{
        borderRadius: '50%',
        width: '2.375rem',
        height: '2.375rem',
        padding: 0,
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        margin: 0,
        // background: rgba(colors.black, 0.05),
        // color: colors.black,
        // '@media (hover: hover)': {
        //   ':hover': {
        //     color: colors.white,
        //     background: colors.primary,
        //   },
        // },
      }}
      onClick={(e) => {
        setColorMode(colorMode === 'default' ? 'dark' : 'default')
      }}
    >
      {colorMode === 'default' ? (
        <FiMoon 
          css={{
            display: 'flex',
            alignItems: 'center',
            justifyContent: 'center',
            margin: '0',
            width: '1.25rem',
            height: '1.25rem',
          }} />
      ) : (
        <FiSun
          css={{
            display: 'flex',
            alignItems: 'center',
            justifyContent: 'center',
            margin: '0',
            width: '1.25rem',
            height: '1.25rem',
          }} />
      )}
    </Button>
  )
}

export default ThemeToggler
```
