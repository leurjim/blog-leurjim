[Seimetel](https://www.seimetelperu.com/) proyecto creado para una empresa de Infraestructura de Telecomunicaciones. El cliente queria darse a conocer a tráves de la Web.

## Como se realizó el proyecto

En este caso el proyecto requería, además de desarrollar la web, realizar el diseño. Por lo cual, se tuvo que recurrir al curso de [**Design for Developers**](https://frontendmasters.com/courses/design-for-developers/) de [Sarah Drasner](https://frontendmasters.com/teachers/sarah-drasner/) profesora en [Frontend Masters](https://frontendmasters.com/dashboard/). Este curso nos brindó el conocimiento y la técnica para realizar la parte estética de la página.

Después de tener el diseño pasamos a la parte del desarrollo. Como es una página informativa se decidió desarrollarla en Gatsby. Para esta parte utilizamos los conocimientos adquiridos en el curso de [Introduction to Gatsby, v2](https://frontendmasters.com/courses/gatsby-v2/) de [Jason Lengstorf](https://www.learnwithjason.dev/) profesor en [Frontend Masters](https://frontendmasters.com/dashboard/) y instructor en [Egghead](https://egghead.io/).

```Js
// Crea un sitio Gatsby

npm init gatsby
```

```Js
// src/pages/index.js

import * as React from "react";

export default function IndexPage() {
  return (
    <main>
      <h1>Hola Seimetel!</h1>
    </main>
  );
}
```
Guarde este archivo, luego inicie el servidor de desarrollo de Gatsby.

```Js
gatsby develop
```

## Optimizar imágenes estáticas

Una de las características más poderosas de Gatsby es su capacidad para procesar imágenes. Esto significa que podemos agregar imágenes en los sitios de Gatsby y se optimizarán automáticamente: marcadores de posición elegantes, imágenes receptivas, carga diferida y más, todo con una configuración mínima.

Para agregar optimización de imagen, debemos agregar el complemento de imagen Gatsby y soporte para Sharp, que es una biblioteca de procesamiento de imágenes de código abierto.

```Js
npm install gatsby-plugin-image gatsby-plugin-sharp gatsby-transformer-sharp
```

## Responsive Web

Para el caso de la adaptación de la web a los diferentes tamaños de pantalla utilizamos lo aprendido en el curso [Master Gatsby](https://mastergatsby.com/) de [WesBos](https://wesbos.com/).

```Js

// layout.module.css

	color: var(--white);
	/* needed for the floated layout */
	clear: both;
	padding: 0;

	height: 90px;

	background: var(--orange);
	color: var(--white);
}

/* navigation */
```
